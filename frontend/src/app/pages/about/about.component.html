About PokéArchive

PokéArchive is a website that lets users browse Pokémon TCG cards and save their favorites for quick access. The site uses the TCGdex API (https://tcgdex.dev ) to display official card data and images.

Developer

The project is developed by Goschad as a first Angular project, created to learn how to work with APIs, build components, and design a clean modern interface.
